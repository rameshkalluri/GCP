# Day 3 — Compute Engine (Virtual Machines)

Notes for Day 3 of the [10-Day GCP Learning Plan](../GCP-10-Day-Learning-Plan.md).

> **Big-picture analogy:** A **Virtual Machine (VM)** is like **renting a computer
> in Google's data center** instead of buying a physical one. You pick how powerful
> it is, what operating system it runs, and you pay only while it's switched on —
> like renting a car instead of buying it.

## Topics
1. [Compute Engine Basics](01-compute-engine-basics.md) — what a VM is, machine types.
2. [Images & Disks](02-images-and-disks.md) — operating systems & storage for VMs.
3. [Spot/Preemptible VMs & Pricing](03-spot-vms-and-pricing.md) — cheaper VMs, discounts.
4. [Metadata, Startup Scripts & Instance Groups](04-metadata-startup-and-groups.md) — automation & scaling.

## Day 3 Goals
- [ ] Explain what a VM is and how to size one (machine types).
- [ ] Understand images (OS) vs disks (storage).
- [ ] Know when to use Spot VMs and how discounts work.
- [ ] Understand startup scripts and instance groups + autoscaling.
- [ ] Launch a VM, SSH in, and install a web server.
