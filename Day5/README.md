# Day 5 — Storage Options

Notes for Day 5 of the [10-Day GCP Learning Plan](../GCP-10-Day-Learning-Plan.md).

> **Big-picture analogy:** Storage is **where you keep your stuff**. Different stuff
> needs different storage — like a home has a **garage** (big boxes you rarely open),
> a **wardrobe** (clothes you use daily), and a **fridge** (fast access to fresh food).
> GCP gives you the right "container" for each kind of data.

## Topics
1. [Cloud Storage (Buckets)](01-cloud-storage-buckets.md) — the warehouse for files/objects.
2. [Storage Classes & Lifecycle](02-storage-classes-and-lifecycle.md) — hot vs cold shelves, auto-tidying.
3. [Disks & Filestore](03-disks-and-filestore.md) — hard drives for VMs & shared folders.
4. [Choosing the Right Storage](04-choosing-the-right-storage.md) — a simple decision guide.
5. [Encryption & Retention](05-encryption-and-retention.md) — key management (CMEK/KMS) & retention/holds/Bucket Lock.

## Day 5 Goals
- [ ] Understand what object storage (buckets) is and when to use it.
- [ ] Know the storage classes (Standard/Nearline/Coldline/Archive) and lifecycle rules.
- [ ] Understand Persistent Disks vs Filestore and when to use each.
- [ ] Pick the right storage type for a given need.
- [ ] Create a bucket, upload/download files, set a lifecycle rule, host a static site.
